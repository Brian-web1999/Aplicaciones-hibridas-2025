# Git

- Configurar el usuario
``` bash
    git config --global user.name "Brian"
    git config --global user.email "brian.gonzalez@davinci.edu.ar"
```

- Inicio el repositorio
``` bash
git init
```
- Agrego los archivos

``` bash
    git add .
```

- Realizo el commit

``` bash
    git commit -m "Semana 02 - Modulos"
```
